AI Assistant
count errors in my log file
To count the number of ERROR occurrences in your log file: 
cat large_log.txt | grep "ERROR" | wc -l
This pipes the file through grep to find ERROR lines, then counts them with wc.
backup my documents
Here's how to create a compressed backup of your Documents folder: 
tar -czf backup.tar.gz ~/Documents
This creates a gzipped tar archive of your Documents directory.
check running node processes
To see all running Node.js processes: 
ps aux | grep node
This will show you all processes with "node" in the command.
